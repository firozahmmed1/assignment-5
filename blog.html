1.What is DOM and what is the purpose of DOM?

DOM means Document Object Model. It's a programming interface using can change document structure, styles, and content.
The Main purpose of DOM is modifying attributes, classes, and styles.

3.What is event bubble?

Event Buble is similar to DOM. A bubbling event goes from the target element straight up. That is, their travel stops after the target phase.
